Laptops

Una laptop, también conocida como ordenador portátil, es un tipo de computadora personal diseñada para ser compacta, ligera y fácil de transportar. A diferencia de las PC de escritorio, las laptops integran todos sus componentes en un solo dispositivo, incluyendo pantalla, teclado, batería y trackpad.
