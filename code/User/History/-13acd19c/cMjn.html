Personal computer

Un Personal Computer (PC) es un computador de uso personal diseñado para realizar diversas tareas, como navegación en internet, edición de documentos, juegos, diseño gráfico, programación y más. Se caracteriza por ser un dispositivo versátil y accesible para individuos y empresas.
